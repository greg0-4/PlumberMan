## PlumberMan
This is a remake of the original Mario Bros. game for TempleOS, game doesnt contain all the features but contains
the basic gameplay loop


## How to install it?
Go to the Releases tab and download the ISO.C file,
then mount the file in TempleOS using <b>Mount();</b> and follow the instructions.
Installing is optional as you can just run the game on the CD.

Alternatively, you can use <a href="https://github.com/aiwnios/Aiwnios">Aiwnios</a> to play the game on all platforms.

## Controls
Arrow keys to move around (including jumping)
...And obviously space to start
